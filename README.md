# Linux Lab Assignment

Graded lab assignment covering Linux fundamentals — environment verification, permissions, links/inodes, file I/O, and storage assessment.

## Structure

- **Q1_Environment/** — User/environment verification (whoami, id, groups, shell, pwd, network connectivity)
- **Q2_Workspace/** — Secure project workspace setup (directory structure, chmod, umask, ownership)
- **Q3_Links/** — Hard link vs symbolic link behavior and inode analysis
- **Q4_IO/** — File access, I/O redirection, and file descriptor investigation
- **Q5_Storage/** — Storage health assessment (disk usage, inode utilization, recommendations)

Each folder contains:
- A report `.txt` file with commands run, outputs, and observations
- Screenshots of terminal execution
- Any files created during the exercise

## Environment

All commands were run in Google Cloud Shell (Debian-based Linux environment).
